# Daros Metaal

Website voor **Daros Metaal** — specialist in fijnmechanische onderdelen en seriematig draai- en freeswerk, gevestigd in Valkenswaard.

## Over de site

Eén-pagina website (`index.html`) gebouwd met Tailwind CSS. Volledig zelfstandig: gewoon `index.html` openen in een browser.

### Functies

- Responsief ontwerp met mobiel uitklapmenu
- **Thema-schakelaar** (rechtsonder): wissel live tussen een licht en een donker design; de keuze wordt onthouden
- Soepel scrollen met actieve navigatie-markering
- Contactformulier met validatie (opent de mailclient via `mailto:`)
- Secties: hero, over, diensten, machinepark, portfolio en contact

## Lokaal bekijken

Open `index.html` rechtstreeks in je browser, of start een lokale server:

```bash
python3 -m http.server
```

Ga daarna naar http://localhost:8000
